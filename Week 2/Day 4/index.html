INDEPENDENT IT CONSULTANT

The purpose of IT in modern Business

1.1 In this modern age of cyber attacks on both big and small companies having a good IT infrastructure in place is vital. With recent company data breaches from the likes of Facebook, Google and others its important to have strong regulations in place to protect users data stored by software and on devices, especially when access to cameras or microphones have been granted.

How to complete a successful digital projects

1.2  Have a clear understanding of the scope of the project by getting as much info as possible. Know the purpose and the target audience the projects geared towards. As for tools, I’d recommend using something standard you and others are comfortable with, unless specifically requested by a client. Do your research, do mockups, set out a budget for things needed and get the clients approval. If you create a timeline for tasks and factor in additional time for testing before agreeing a deadline and costs, this will aid in completing a successful project. It also helps if you have the patience of a saint.

Planning and carrying out task using IT

1.3 Trello for ease and flexibility to plan your website. Visual Studio Code to combine your HTML, CSS and JavaScript files. And a GitHub Repository to share your files.

Risks that might impact a digital project

1.4 Project management is so important. Plan each task and allocate enough time needed to complete those tasks. Not allowing yourself enough time to run numerous test on different platforms to work out bugs causes digital projects to fail.

Using IT systems and software

1.5 It’s important to select an IT system and software that’s supported and used by everyone like MySQL to prevent problems that can arise by using something that’s just been released, as the last thing you need is to get stuck.

The outcome of chosen technology

1,6 MySQL is the world's most popular open-source database as it runs on virtually all platforms, including Windows, UNIX and Linux. It’s chosen over other’s because it’s the most trusted and the most widely used in the challenges of next generation web, cloud, and communications services.
